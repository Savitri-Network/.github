# Brand Guidelines

> Usage rules for Savitri Network brand assets.

## Logo Usage

### Clear Space
Maintain minimum clear space around the logo equal to the height of the "S" in Savitri.

### Minimum Size
- Digital: 120px width minimum
- Print: 25mm width minimum

### Do
- Use official logo files from the `logos/` directory
- Maintain aspect ratio
- Use on solid backgrounds (dark preferred)
- Use the monochrome version on busy backgrounds

### Do Not
- Stretch, rotate, or distort the logo
- Change logo colors outside the approved palette
- Add effects (shadows, gradients, outlines)
- Place on low-contrast backgrounds
- Recreate or approximate the logo

## Typography

| Use | Font | Fallback |
|---|---|---|
| Headlines | **Inter** (Bold/Semibold) | Helvetica Neue, Arial |
| Body text | **Inter** (Regular) | Helvetica Neue, Arial |
| Code / technical | **Fira Code** | Source Code Pro, Courier New |
| Monospace UI | **JetBrains Mono** | Fira Code |

## Tone of Voice

| Context | Tone | Example |
|---|---|---|
| Technical docs | Clear, precise, no jargon shortcuts | "The mempool scores transactions using SIMD intrinsics" |
| Marketing | Confident, not hype | "50K TPS with 150ms finality" not "revolutionary next-gen" |
| Community | Friendly, direct | "Run a node in 60 seconds" |
| Security | Serious, factual | "Report vulnerabilities to security@savitri.network" |

### Writing Rules
- Use active voice
- Lead with the outcome, not the process
- Numbers over adjectives ("150ms" not "ultra-fast")
- No superlatives without data to back them up
- American English for all public-facing content

## Badge Style

Shields.io badges use these parameters:

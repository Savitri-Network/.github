## Dark / Light Mode

- **Dark mode** (default): Dark Base (#1a1a2e) background, white (#e0e0e0) text
- **Light mode**: White (#ffffff) background, Dark Base (#1a1a2e) text
- Savitri Red works on both backgrounds without modification
